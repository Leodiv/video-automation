2. Description du service

FlowPublish est une plateforme permettant aux créateurs de contenu de publier automatiquement des vidéos courtes sur TikTok via l'API officielle TikTok Content Posting. Le service utilise les autorisations TikTok suivantes :

— video.upload : Pour uploader des vidéos sur votre profil TikTok

— video.publish : Pour publier directement les vidéos sur votre profil

— user.info.basic : Pour identifier l'utilisateur connecté
